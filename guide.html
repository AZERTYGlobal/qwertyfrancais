Guide rapide

?? Les 5 Principes Clés

1. É en accès direct

La touche ; (sous le petit doigt droit) est remplacée par é. C'est la seule lettre "non-QWERTY" ajoutée en accès direct, car c'est la plus fréquente.

?? Pour faire un point-virgule ;, faites Maj + ,.

2. Verrouillage majuscule intelligent

Activez Caps Lock et tapez sur une lettre ou un accent : vous obtenez une majuscule ! é ? É, à ? À, ç ? Ç.

Les touches chiffres ne sont pas affectées par Caps Lock (elles restent des chiffres). Idéal pour saisir des adresses ou des variables (ex: VARIABLE_1).

3. Tous les accents sous la main

Les touches à droite du P et du L deviennent des touches mortes "magiques" :

• [ devient ` (grave) ? tpapez ` puis e = è
• ) (Shift+0) devient ^ (circonflexe) ? ^ puis o = ô
• ' devient ' (apostrophe) ou ´ (aigu) selon le contexte.

4. Symboles de code INTROUVABLES ? Non !

Ne paniquez pas. Les symboles sont déplacés sur la rangée de repos pour un accès ultra-rapide avec AltGr :

AltGr + D / F = { }
AltGr + J / K = [ ]
AltGr + H = |
AltGr + G = \

5. Un Ç facile à retenir

Le c cédille se trouve logiquement sur AltGr + C. Impossible à oublier. Pour la majuscule Ç, faites simplement Caps Lock + AltGr + C (ou juste AltGr + Maj + C).
