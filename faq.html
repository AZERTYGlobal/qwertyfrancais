🔠 Pourquoi QWERTY Français ?

Pourquoi pas juste US International ?

Le layout US International oblige à utiliser des touches mortes pour les guillemets (' et "). C'est très pénible pour la programmation.

QWERTY Français garde les guillemets instantanés (comme sur US Standard) mais offre un accès facile aux accents français via des raccourcis intuitifs.

✒️ Accents & Typographie

Comment faire É, È, À, Ç ?

Tout est pensé pour être rapide :

  • É : Touche directe (à la place du point-virgule) ou via accent aigu.
  • Accents graves (à, è, ù) : Touche morte grave (D11 base) puis lettre.
  • Cédille (ç) : AltGr + C.
  • Circonflexe (ê) : Touche morte circonflexe (D12 base) puis lettre.
Comment faire les guillemets français « » ?
  • AltGr + Z → «
  • AltGr + X → »

Z et X sont en bas à gauche, facile d'accès !

🌍 Langues européennes

QWERTY Français supporte-t-il d'autres langues (espagnol, allemand, etc.) ?

Oui ! QWERTY Français permet de taper facilement en :

  • 🇪🇸 Espagnol : ñ, ¿, ¡
  • 🇩🇪 Allemand : ä, ö, ü, ß
  • 🇵🇹 Portugais : ã, õ, ç
  • 🇮🇹 Italien : à, è, ì, ò, ù
  • 🇩🇰🇸🇪 Nordiques : æ, ø, å
  • 🇵🇱🇨🇿🇭🇺🇷🇴🇹🇷 Et bien d'autres...

⌨️ Symboles courants

Comment faire @ (arobase), [ ], { } facilement ?

Avec QWERTY Français :

  • @ : Shift + 2 (comme sur QWERTY US)
  • [ ] : AltGr + J / K (rangée de repos)
  • { } : AltGr + D / F (rangée de repos)

❓ Questions générales

J'ai un clavier AZERTY physique, ça marche ?

Ça fonctionnera techniquement, mais ce que vous taperez ne correspondra pas à ce qui est écrit sur vos touches.

Ce layout est conçu pour les claviers physiques QWERTY (ANSI) ou pour ceux qui connaissent le QWERTY par cœur et tapent sans regarder.

Combien de temps pour s'adapter ?

La plupart des utilisateurs s'adaptent en quelques minutes à quelques heures. Il n'y a que 5 changements majeurs à mémoriser.

Moins de 1% de vos frappes seront différentes ! Les lettres et chiffres ne bougent pas.

Dois-je acheter un nouveau clavier ?

Non, absolument pas !

QWERTY Français est une solution purement logicielle. Elle s'installe sur votre Windows et redéfinit ce que font vos touches actuelles. Votre matériel existant est 100% compatible.

Est-ce compatible avec les jeux vidéo (ZQSD) ?

Oui et non. C'est un clavier QWERTY, donc on utilise WASD pour se déplacer.

Si vous basculez Windows sur ce clavier, vos jeux le verront comme un clavier anglais standard. Généralement, les jeux détectent la position physique des touches (Scan Codes), donc WASD fonctionnera naturellement à la place de ZQSD.

Comment le mettre par défaut ?

Dans les paramètres de langue Windows, supprimez les autres claviers (Français AZERTY, Anglais US) une fois que vous êtes à l'aise avec celui-ci. Cela évitera à Windows de changer de langue tout seul.

Disponible sur Mac et Linux ?

Pas encore ! La version actuelle est pour Windows 10 et 11 uniquement. Les versions macOS et Linux sont prévues pour plus tard.

Comment désinstaller ?

Via les paramètres de langue Windows (Supprimer le clavier), ou via "Ajouter ou supprimer des programmes" si vous avez utilisé l'installateur EXE.
